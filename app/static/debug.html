Token Pairing

Request a new authentication token from the TV. This will trigger a pairing prompt on the TV screen that you must accept.

Instructions:

  1. Ensure the TV is fully ON (not in standby mode)
  2. Click "Request New Token" below
  3. Check the TV screen for a pairing prompt
  4. Use the TV remote to select "Allow"
  5. The new token will be displayed here
